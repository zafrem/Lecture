# Basic Security Concepts

This directory contains Python implementations of fundamental cybersecurity concepts. Each script is designed to demonstrate a specific area of security.

## Contents

1.  **[Authentication & Access Control](1_authentication.py)**
    - Password hashing with bcrypt and Argon2.
    - Multi-Factor Authentication (MFA) simulation.
    - JWT (JSON Web Token) generation and validation.
    - Role-Based Access Control (RBAC) implementation.

2.  **[Encryption](2_encryption.py)**
    - Symmetric encryption (AES).
    - Asymmetric encryption (RSA).
    - Secure key management practices.

3.  **[Network Security](3_network_security.py)**
    - Socket security and SSL/TLS.
    - Port scanning and service detection.
    - Basic firewall rules and packet filtering concepts.

4.  **[Vulnerability Assessment](4_vulnerability_assessment.py)**
    - Simple vulnerability scanning logic.
    - Identifying common misconfigurations.
    - OWASP Top 10 related checks.

5.  **[Logging & Monitoring](5_logging_monitoring.py)**
    - Secure logging practices.
    - Real-time event monitoring.
    - Alerting mechanisms for suspicious activities.

6.  **[Infrastructure Security](6_infrastructure_security.py)**
    - System hardening scripts.
    - Permission and ownership checks.
    - Environment variable security.

7.  **[Incident Response](7_incident_response.py)**
    - Forensic data collection.
    - Automated containment actions.
    - Log analysis during an active incident.
